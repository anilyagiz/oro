## Appendix: GRAIL API Reference

### Endpoints
1. **POST /users** - Create user (KYC hash + wallet)
2. **POST /purchases** - Create purchase transaction
3. **GET /quotes** - Get GOLD price quote
4. **GET /transactions/:id** - Get transaction status

### Authentication
- API Key in Authorization header
- Wallet signature for transactions

### Self-Custody Flow
1. Connect wallet
2. Create user (POST /users)
3. Get quote (GET /quotes)
4. Create purchase (POST /purchases)
5. Sign transaction
6. Submit to GRAIL
7. Poll for confirmation
8. Show success/failure
